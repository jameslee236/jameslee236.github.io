Ruby

Loop Structure
for variable [, variable...] in expression
[do]
code
end

'For' Example
for i in 0..5
puts "value of local variable is #{i}"
end

'While' Example
while x >= 0
puts x
x = x -1
end

Iterating through Each
Example:
a = ['a','b','c']
a.each {|n| puts n}
end

Methods
def hello
puts "hello"
end

Hashes
hash_name = {hash_key: hash_value}
Access by using:
hash_name[:hash_key] —-> hash_value

JavaScript

Loop Structure
for (statement 1; statement 2; statement 3) {
code block to be executed
}
Statement 1 - executed before the loop starts.
Statement 2 - defines the condition for running.
Statement 3 - executed each time after the loop runs.

'For..in' Example
var person = {
first_name: "John",
last_name: "Doe",
age: 25
};
var text = ""
var x;
for (x in person) {
text += person[x];
}

Iterating through Each
var a = ['a','b','c'];
a.forEach(function(entry){
console.log(entry);
});

Functions
function hello() {
console.log("hello");
}

Objects
var hash_name = { hash_key: hash_value}
Access by using:
hash_name.hash_key —-> hash_value
